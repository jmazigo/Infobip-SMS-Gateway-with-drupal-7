
Description
===========

This module provides integration between Infobip SMS service and
Drupal SMS framework project.

Dependencies
============
1.Drupal SMS Framework module.
2.Requires Infobip subscription.( https://www.infobip.com )

Installation
============
1.Download and install the module as usual.
2.Go to "admin/smsframework/gateways/infobip_sms_gateway" and
configure your username, password of your Infobip subscription.
